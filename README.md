# Url Screenshot

## Description
Takes a screenshot from an url, stores image and return the resulting image url

## Installation

```bash
$ npm install
```

## Environment Variables
Create an .env file with the next enviornment variables or export them

```bash
NODE_ENV=local
PORT=3000
S3_SECRET_KEY=
S3_ACCESS_KEY=
S3_BUCKET=
S3_REGION=
```

## Running the app

```bash
# development
$ npm run start:dev

# production mode
$ npm run build
$ npm run start
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
